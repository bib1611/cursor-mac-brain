## RULE ZERO — voice wins on conflict
Lisec is generic business-writing advice. Adam's voice is confessional KJV. Where they collide, voice-DNA wins.

- **Fragments / staccato:** Lisec calls them a tell. For Adam they are craft. Keep the fragment that carries a wound. Cut only the EMPTY fragment.
- The test is never "is this a fragment." The test is "does this line carry a wound or a fact, or is it decoration?"
- Where Lisec and voice-DNA AGREE, obey both: no decorative em dashes, no filler, concrete over abstract, start from owned material.

## RULE ONE — source from the wound, never the website
The number-one cause of slop is generating from scraped bio facts. Lisec's own first step is the same: start from OWNED source material — notes, voice dumps, transcripts, real moments.

- Never invent a personal scene. If there is no real moment, write `[INSERT: real moment from Adam]` and STOP. Get it from Adam.
- Facts pulled off the website and flatly stitched read like a bot even when every fact is true.

## THE TELLS — reject or revise any sentence that...

**Adam's Kill List (highlights — full list in ANTI_AI_SLOP_REFERENCE.md):**
delve, unpack, navigate, landscape, foster, leverage, comprehensive, robust, tapestry, resonate, moreover, furthermore, "it's worth noting," "at the end of the day," "let's dive in," "here's the thing," game-changer, holistic, ecosystem.

**Lisec's 6.5 tells:**
1. **Negation framing** ("not X, but Y") → delete the setup, lead with the affirmative claim.
2. **Word-density filler** → compress hard (Lisec: tighten ~85% when the model rambles).
3. **Vague abstraction** → name the concrete thing, person, date, mechanism, consequence.
4. **Rule-of-three reflex** → use the REAL count (two, four, five), not an automatic triad.
5. **Metaphor-as-argument** → argue the subject directly; keep a metaphor only if precise and not carrying the claim.
6. **Prompt-answer opening** (answers an invisible prompt) → first line must stand alone for a cold reader.
6.5. **Surface tics** — em dashes, exclamation points, emojis, decorative punctuation, adverb stuffing → normalize. *(Intentional fragments are EXEMPT per Rule Zero.)*

## THE PASSES — edit word-by-word, never one-shot
1. Source from owned material (Rule One).
2. Strip tells — negation framing, automatic triads, filler.
3. Add precision — names, dates, objects, consequences. Say the name, not "people."
4. Repair logic and sequence — each line follows the last; line one stands alone.
5. Discipline metaphors — cut any metaphor doing the job of proof.
6. Restore human judgment — the wound, the reversal, the line a generic model cannot supply.
7. Surface polish — punctuation and adverbs. KJV only. Keep Adam's intentional fragments.
