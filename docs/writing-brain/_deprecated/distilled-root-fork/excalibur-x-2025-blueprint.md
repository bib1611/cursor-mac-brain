## Core thesis
Grow on X by working ten gates in order: qualify the account, organize your inputs, define your purpose, build real relationships, hit the paid thresholds, hype your presence, build owned assets, create offers, set yearly targets. The sequence is the product. Most accounts skip the qualify and threshold gates and stall.

## Key frameworks / rules

### The 10-gate yearly roadmap (net-new as a named ladder)
Run these in order. Each gate unlocks the next.
1. Qualify. Get verified. Two-factor on. No spam. No links in body. Raise TweepCred.
2. Organize. Find mentor accounts, subscribe to them, build a content library, build and follow lists, write everything down, set up a notes system.
3. Define purpose. Write an account-definition post. Pin it or highlight it on the profile.
4. Build relationships. Subscribe plus notifications on for targets. Comment, post, attend Spaces, DM one new person a day, make friends.
5. Get paid. Pass the threshold gates (see Numbers). Qualify for ad revenue sharing.
6. Hype. Weekly Spaces. Threads twice a week. Share and support inside communities. Build networking groups.
7. Build. Email list, landing page, lead magnet, premium-follower count, apply for subscriptions.
8. Create. Low-ticket offer, sell, listen to feedback, collect testimonials, then a high-ticket offer.
9. Define yearly goals. Follower target, revenue target, email-subscriber target, life and business goals.
10. Mentorship. Apply to the group program.

### The account-definition post (net-new artifact)
A single pinned or highlighted post that states exactly what the account is and who it serves. Step 3 of the ladder. The profile's anchor, written before relationship-building starts so new visitors know in one read whether to follow.

### Qualify-first rule
Before reach work, lock the account hygiene: verified, two-factor, zero spam behavior, no links in post bodies, rising TweepCred. Reputation is the gate, not an afterthought.

### Organize-your-inputs system
A named pre-content stack: mentor list, subscriptions to mentors, a saved content library, follow-lists by niche, a write-everything-down habit, a notes system. Build the intake before the output.
